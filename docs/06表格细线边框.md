### 表格标签table

```
<table  align="center" width="400" height="200" border="1" cellspacing="0">

<!--

    align：属性，表格的位置
        center:属性值，居中
        left：属性值，居左
        right：属性值，居右

    width:属性,表格的长度。
        属性值：数值类型
    
    height:属性，表格的高度
        属性值：数值类型

    border:属性，表格边框的粗细。
        属性值：数值类型。
    
    cellspacing:属性,规定单元格与单元格之间的空间距离。
        属性值：数值类型。
        
-->
```

#### 表格添加边框样式

```
table,td{
    border: 1px solid pink;
}
```

#### 表格的细线边框属性

==border-collapse==，属性，控制浏览器绘制表格边框的方式，控制相邻单元格的边框是否合并。

```
border-collapse: collapse;
表示相邻的边框合并在一起。
```

collapse：合并的意思
